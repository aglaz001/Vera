# Enhanced Prompt: Report Preview & Export

Final report preview page for the Vera ESG platform.

**DESIGN SYSTEM (REQUIRED):**
- Platform: Web, Desktop-first
- Tone: Calm Authority, Editorial Archivist
- Palette: 
    - Forest (#1A3D22) for headings and primary actions
    - Grove (#2E6B40) for indicators
    - Fern (#5AAF72) for positive growth accents
    - Mist (#EAF5ED) for subtle section fills
    - Linen (#F7F3EC) for the main off-white background
- Typography: 
    - Instrument Serif (Headings, Editorial asides in italics)
    - DM Sans (UI Labels, Body)
- Styles: 14px card radius, no 1px borders (use tonal washes), pill-shaped buttons.

**Page Structure:**
1. **Header:** Minimal navigation breadcrumb: "Reports > 2024 ESG Annual Report".
2. **Preview Canvas:** Large centered area showing a high-fidelity scrollable report preview. Layout matches a premium printed document.
3. **Actions Sidebar (Right):**
    - "Export Options" group: Branded PDF download, Editable DOCX, Shareable Link.
    - "Report Settings" group: Toggle GRI Content Index, Toggle ESG Score Breakdown.
4. **Branding Panel (Bottom):** Logo upload slot and accent color picker for the final PDF.

# Enhanced Prompt: Settings & Profile

Organization and team management page for the Vera ESG platform.

**DESIGN SYSTEM (REQUIRED):**
- Platform: Web, Desktop-first
- Tone: Calm Authority, Professional Expert
- Palette: Same as above.
- Typography: Same as above.
- Styles: Same as above.

**Page Structure:**
1. **Header:** "Settings & Configuration".
2. **Tabbed Navigation:** Organization Profile, Team Members, Billing, Frameworks.
3. **Content Area:** 
    - **Organization Profile:** Forms for company name, sector, and ESG primary contact.
    - **Team Members:** Table with Name, Role (Admin, Editor), and "Invite Member" button.
    - **Frameworks:** Selection list for active reporting standards (GRI, CSRD, TCFD).
4. **Footer:** Standard links and version info.
